# SEO Research — Dry Vent Tampa

This file is populated when you run `/firecrawl` inside Claude Code. It captures:

- Top competitor analysis for "dryer vent cleaning Tampa" and variants
- Content gaps vs our current coverage
- Keyword opportunities not currently targeted
- Schema.org implementations on competitor sites
- FAQ questions competitors answer that we don't

## How to generate this report

Open Claude Code in this project:

```bash
cd dry-vent-tampa
claude
```

Then run:

```
/firecrawl
```

The command is defined in `.claude/commands/firecrawl.md` and will use the Firecrawl MCP server to scrape competitors, then write the full report here.
